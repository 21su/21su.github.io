주요 기능


  1. 비회원도 월드컵을 진행할 수 있지만 댓글을 남기거나 제작/수정이 불가능

  2. 회원가입/로그인 기능으로 회원상태로 진행할 수 있으며, 회원은 월드컵 제작 및 자신이 만든 작품 수정 가능

  3. admin 계정을 통해 의도가 벗어난 글은 삭제 및 회원 제재 가능


상세 기능


1. 메인 페이지 기능


  - 최근에 추가된 순서대로 공개 상태인 월드컵을 리스트에 출력함.

  - 리스트는 첫번째와 두번째 사진을 메인으로 걸어두고 월드컵 이름 설명 등을 출력

  - 리스트 상단에 검색 기능이 있음

  - 시작 버튼을 통해 해당 월드컵으로 이동


2. 월드컵 제작 기능


  - 회원 상태일 경우에만 사용 가능하며 제목과 월드컵 설명 및 공개 여부를 정할수 있음

  - 그 후 사용할 이미지파일을 업로드 하고 이미지 수정 탭으로 넘어가 이미지를 변경, 삭제및 이름을 수정할 수 있음


3. 내가만든 월드컵


  - 회원 상태일 때 내가 만들었던 월드컵이 출력되고 월드컵시작 및 랭킹확인 월드컵 수정 탭으로 이동하는 버튼과 해당 월드컵을 삭제하는 버튼이 같이 출력됨


4. 내정보 창


  - 내정보 창에서 비밀번호를 재확인 하면, 닉네임과 이메일 변경 가능


5. 메인 월드컵 기능


  - 월드컵을 시작하면 업로드 된 이미지의 수량에 따라 실행할 라운드 수를 정할 수 있음

  - 라운드 수를 정하면 해당 라운드에 맞는 개수의 이미지를 랜덤으로 2개의 리스트로 나눠서 받아옴

  - 라운드 진행시 2개의 리스트의 같은 인덱스의 이미지를 화면에 출력 유저가 둘중 하나를 선택하면 다음 인덱스의 이미지를 출력 마지막 이미지를 선택후 선택된 이미지를 리스트로 받아서 다시 한번 해당 주소로 호출

  - 선택된 이미지를 랜덤하게 2개의 리스트로 다시 나눠 담음

  - 그렇게 반복 후 마지막 이미지가 남으면 랭킹 테이블에 삽입 후 결과 및 랭킹 확인창으로 넘어감


6. 랭킹 확인창


  - 해당 월드컵에 최종으로 선택된 이미지의 데이터를 기준으로 몇번 선택 되었는지를 비율로 나타내 며 해당 월드컵에 관한 댓글들을 남길 수 있음


7. 어드민 계정 기능


  - 어드민 계정 로그인 시 월드컵의 삭제가 가능해지며, 헤더에 회원관리 창으로 이동할 수 있는 버튼이 활성화 됨

  - 회원관리 창에서는 회원 목록이 출력되며 회원 삭제가 가능함


8. CSS작성


  - 웹 페이지에 사용되는 모든 CSS를 작성함.


기획 및 메인화면 구성

   스프링부트를 이용해서 만든 첫 프로젝트는 개인으로 진행한 프로젝트다. 프로젝트를 기획할 때 흔한게 아닌 특별한 기능을 가진 새로운 도전을 해보고 싶었고, 하나의 테이블 안에 정해지지않은 여러개의 이미지를 연동해 관리해보고 싶다 생각해 심심할때 자주 들어갔던 이상형 월드컵사이트를 만들어 보고자 생각했다.

   메인화면은 데이터베이스에 있는 비공개가 아닌 월드컵들을 가져와 Model을 이용해 리스트로 받아왔고, 해당 리스트를 통해 월드컵 제목, 썸네일, 설명등을 화면에 뿌려주었다.
   검색 input창과 검색설정을 해줄수 있게 하여 제목 또는 작성자를 검색해 원하는 월드컵을 찾을수 있게 했다.
메인화면에서는 로그인/회원가입 창으로 회원관련처리를 할수있으며, 로그인이 되어있을 때 월드컵 만들기 창이나 내 월드컵 관리 창으로 넘어갈 수 있다.

회원 관련

   회원가입시 아이디와 비밀번호가 조건에 맞춰져있는지 확인한 후 Ajax 통해 데이터베이스로 넘어가 해당 아이디가 중복인지를 체크한다. 그 후 중복이 아닐경우 해당 회원가입 정보를 MemberTable 에 담는다.
   로그인 시에는 Ajax를 통해 input에 입력한 MemberDTO의 값이 데이터베이스에 저장해놓은 데이터값과 일치할 경우 해당 정보값을 Session 담아 로그인처리를 한다.

월드컵 제작/수정

   회원가입및 로그인을 했을경우에 월드컵을 제작할 수 있다. 먼저 월드컵에 관한 정보를 Ajax 를 통해 데이터베이스의 BoardTable 저장한 후 다시 넘어와 이미지를 등록시킬 수 있다.
   이미지를 등록할 시에는 한번에 여러장의 이미지를 등록할 수 있고, 이미지를 등록할 때 해당 이미지정보를 Service 로 가져가 for문 을 통해 해당 이미지들을 하나씩 해당 이미지 관리 폴더에 저장하며 ImageTable 에 등록시키는데, 해당 ImageTable은 BoardTable 과 연동되어 있다.
   이미지 등록을 끝냈을 경우 위의 이미지 이름 수정/삭제 탭으로 넘어가면 해당 월드컵에 등록되어있는 이미지 정보가 나열된다. 원하는 이미지의 input창에 원하는 이미지의 이름을 넣고 엔터키를 누르면 Ajax를 통해 해당 이미지가 저장되어있는 데이터의 이름값을 수정한다.
   또한 이미지 변경칸을 통해 해당 이미지를 수정할 수 있으며, 삭제 버튼을 통해 해당 이미지의 데이터를 삭제시킬 수 있다.

월드컵 플레이

   원하는 월드컵을 플레이하면 먼저 해당 월드컵에 저장되어있는 이미지의 수량을 기준으로 월드컵의 최대 라운드 수를 정한다.
   그 후 진행할 라운드수를 정할 수 있으며, 라운드 수를 정한 후 실행시키면 해당 라운드에 따라 해당 월드컵의 이미지를 그 수만큼 랜덤으로 가져와 2개의 리스트에 담아서 Model을 통해 jsp에 넘긴다.
  
   라운드가 진행되면 Controller에서 가져온 2개의 리스트의 데이터값을 각각 하나씩 양옆으로 화면에 출력한다. 출력된 두개의 이미지중 하나를 클릭하면 해당 이미지가 선택되어 다음 리스트에 저장이되고, 두 리스트의 이미지가 모두 선택 되었을 경우 따로 저장해놓은 해당 리스트를 다시 해당 jsp에 호출해 2개의 리스트로 랜덤하게 다시 담고 라운드를 진행한다.
   일련의 과정을 이미지 하나가 남을 때까지 반복하고 이미지 하나가 남았을 때 해당 이미지를 최종선택으로 파악 후 해당 이미지를 rankTable에 넘긴다.
  
   모든 과정이 완료되면 랭킹페이지로 넘어가 해당 월드컵의 선택빈도를 기준으로 이미지의 선호도가 표시되고 해당 월드컵에 자신의 생각이나 건의사항같은걸 넣을 수 있게 댓글 기능을 추가해 주었다.
